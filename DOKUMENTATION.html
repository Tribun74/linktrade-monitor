Übersicht

Der Linktrade Monitor ist ein WordPress-Plugin zur professionellen Verwaltung und Überwachung von Backlinks. Es unterstützt drei Kategorien:

Linktausch

Gegenseitige Verlinkungen mit Partnern. Das Plugin überwacht sowohl den eingehenden als auch den ausgehenden Link und berechnet einen Fairness-Score.

Linkkauf

Bezahlte Backlinks mit Laufzeitverwaltung, Kosten-Tracking und automatischen Ablauf-Erinnerungen.

Kostenlos

Gratis erhaltene Backlinks (Gastbeiträge, Verzeichnisse, etc.) mit Status-Monitoring.

Kernfunktionen

  • Automatische Link-Überprüfung (HTTP-Status, nofollow, noindex, sponsored)
  • Gegenseitigkeits-Tracker für Linktausch-Partner
  • Fairness-Score-Berechnung
  • Ablauf-Erinnerungen per E-Mail
  • Kontakt-Historie für Partner-Kommunikation
  • Check-Historie mit Verlauf
  • Dashboard mit Übersicht und Handlungsbedarf

Installation

Systemvoraussetzungen

  • WordPress 6.0 oder höher
  • PHP 8.0 oder höher
  • MySQL 5.7+ oder MariaDB 10.3+

Installationsschritte

  1. Plugin-Ordner linktrade-monitor nach /wp-content/plugins/ kopieren
  2. Im WordPress-Admin unter "Plugins" das Plugin aktivieren
  3. Das Plugin erstellt automatisch die benötigten Datenbanktabellen
  4. Unter "Linktrade" im Admin-Menü die Einstellungen konfigurieren
Hinweis: Bei der Aktivierung werden 5 Datenbanktabellen erstellt und die Standard-Optionen gesetzt.

Funktionen im Detail

Dashboard

Das Dashboard zeigt eine Übersicht mit:

  • Statistik-Karten: Gesamtzahl Links, Online-Status, Warnungen, Offline
  • Neueste Links: Die 5 zuletzt hinzugefügten Links mit Quick-Actions
  • Ablaufende Links: Links die in den nächsten 30 Tagen ablaufen

Link-Prüfung

Die Link-Prüfung analysiert:

Prüfpunkt Beschreibung Status bei Problem
HTTP-Status Erreichbarkeit der Seite (200, 301, 404, etc.) Offline
Link vorhanden Ist der Backlink auf der Seite? Offline
nofollow Hat der Link rel="nofollow"? Warning
noindex Hat die Seite meta noindex? Warning
sponsored Hat der Link rel="sponsored"? Info

Fairness-Score (nur Linktausch)

Der Fairness-Score bewertet die Gegenseitigkeit:

Score Situation
100 Beide Links online und dofollow
60 Mein Link dofollow, Partner nofollow
50 Beide Links offline
0 Mein Link online, Partner offline

Alerts-Tab

Der Alerts-Tab zeigt alle Links die Aufmerksamkeit brauchen:

  • Links mit Status "warning" oder "offline"
  • Linktausch-Partner mit unfairer Gegenseitigkeit
  • In den nächsten 30 Tagen ablaufende Links

AJAX API

Sicherheit: Alle AJAX-Endpoints erfordern einen gültigen Nonce (linktrade_nonce) und Admin-Berechtigung.

Verfügbare Endpoints

linktrade_get_links

Holt eine Liste aller Links mit optionalen Filtern.

POST /wp-admin/admin-ajax.php
action: linktrade_get_links
nonce: {nonce}
category: tausch|kauf|kostenlos (optional)
status: online|warning|offline|unchecked (optional)
search: Suchbegriff (optional)

linktrade_get_link

Holt einen einzelnen Link für die Bearbeitung.

POST /wp-admin/admin-ajax.php
action: linktrade_get_link
nonce: {nonce}
id: Link-ID

linktrade_save_link

Speichert einen neuen Link oder aktualisiert einen bestehenden.

POST /wp-admin/admin-ajax.php
action: linktrade_save_link
nonce: {nonce}
id: Link-ID (0 für neu)
partner_name: Name
partner_contact: Kontakt
category: Kategorie
...weitere Felder

linktrade_delete_link

Löscht einen Link.

POST /wp-admin/admin-ajax.php
action: linktrade_delete_link
nonce: {nonce}
id: Link-ID

linktrade_check_link

Prüft einen Link manuell. Bei Linktausch werden automatisch beide Richtungen geprüft.

POST /wp-admin/admin-ajax.php
action: linktrade_check_link
nonce: {nonce}
id: Link-ID

linktrade_save_contact

Speichert einen Kontakt-Eintrag.

POST /wp-admin/admin-ajax.php
action: linktrade_save_contact
nonce: {nonce}
link_id: Link-ID
contact_type: email|phone|meeting|other
direction: incoming|outgoing
subject: Betreff
content: Inhalt

Cron-Jobs

Hook Intervall Funktion
linktrade_check_links Wöchentlich Prüft alle Links in Batches
linktrade_check_reminders Täglich Sendet Ablauf-Erinnerungen per E-Mail
linktrade_check_fairness Täglich Warnt bei unfairen Linktausch-Partnern
Batch-Verarbeitung: Der Link-Check verarbeitet standardmäßig 50 Links pro Durchlauf mit 3 Sekunden Pause zwischen Requests, um Server nicht zu überlasten.

Konfiguration

WordPress-Optionen

Option Standard Beschreibung

linktrade_email_notifications true E-Mail-Benachrichtigungen aktiviert
linktrade_notification_email admin_email E-Mail-Adresse für Benachrichtigungen

Link-Checker Einstellungen

Der Link-Checker in class-link-checker.php verwendet:

  • User-Agent Rotation: 6 verschiedene Browser-User-Agents
  • Timeout: 15 Sekunden pro Request
  • Redirects: Bis zu 5 Weiterleitungen werden gefolgt
  • URL-Matching: Flexible Erkennung (http/https, www/non-www, mit/ohne trailing slash)
PHP 8.2+ Kompatibilität: Der HTML-Parser verwendet die UTF-8 XML-Declaration Methode statt dem deprecated mb_convert_encoding().
